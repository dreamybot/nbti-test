# NBTI 职场人格测试 × AI 多Agent 求职修炼系统

## 作品简介

NBTI（Naive Behavioral Type Indicator）是一款基于原创人格体系的职场人格测试与AI求职辅助系统。作品由两部分构成：前端为30道情境题驱动的职场人格测试，输出27种互联网语感人格类型及15维度雷达图谱；后端为三个AI Agent（简历优化、模拟面试、岗位情报）组成的求职修炼系统。人格数据驱动Agent服务，形成"认知→提升"闭环。

---

## 市场调研

### 行业现状

当前大学生求职市场存在三个断层：

1. **人格测试与求职脱节**：MBTI、DISC等工具流行但止步于"认识自己"，无法落地到简历、面试等实操环节。
2. **AI求职工具缺乏引导**：市场上简历优化、模拟面试工具众多，但用户需要先了解自己适合什么，再使用工具。
3. **严肃工具门槛高**：传统职业测评量表枯燥、传播性差，年轻用户更偏好有趣、有梗的方式。

### 竞品分析

| 竞品类型 | 代表产品 | 不足 |
|---------|---------|------|
| 传统人格测试 | MBTI、16Personalities | 无求职落地，且用户已审美疲劳 |
| AI简历工具 | 有见求职精灵、智能简历 | 重工具轻认知，缺乏人格维度 |
| AI面试工具 | iFaceOff、HireIQ | 人格特征未融入面试策略 |
| 职业规划平台 | 智联招聘、前程无忧 | 偏信息撮合，缺乏AI个性指导 |

### 市场机会

NBTI瞄准"人格认知→求职行为"之间的空白地带，用趣味测试建立用户连接，用AI Agent承接深度服务，形成差异化竞争力。

## 核心功能及创新点

### 功能一：30道情境题测试

覆盖五大模型×15维度（自我模型、情感模型、态度模型、行动驱力、社交模型），每题3个选项映射L/M/H三级评分。支持隐藏题（爱好选择→触发隐藏人格分支）、防连点锁、选项交错入场动画。

### 功能二：27种职场人格体系

原创人格类型，区别于MBTI/DISC，用互联网语感命名：棋手、大佬、莽夫、躺学家、小趴菜、玻璃体、尖叫机、小喇叭、独狼、打工人、佛星人、自洽体、感恩侠等。每个人格包含code/中文名/tagline/描述/5个推荐岗位/15维L/M/H模式串。

### 功能三：15维度雷达画像

五大模型×15维度得分可视化，Canvas实时绘制。维度包括：职场自信度、职业清晰度、成就驱动力、职场安全感、工作投入度、职场依赖边界、组织信任度、规则遵从度、工作意义感、风险偏好、决策速度、执行风格、职场社交动能、人际边界感、表达策略。

### 功能四：曼哈顿距离匹配算法

基于15维L/M/H向量的曼哈顿距离计算，支持极端冲突加权、兜底非标人人格（相似度<60%触发）。

### 功能五：AI简历优化Agent

用户粘贴目标岗位JD和简历后，AI实时分析JD关键能力点、简历匹配度评分、缺失能力预警、逐段优化建议。支持"AI评分"和"AI优化"两种模式。

### 功能六：AI模拟面试Agent

基于JD实时AI出题（非预设题库），逐题作答后AI评分并给出反馈。支持跳过、进度追踪、平均分统计。

### 功能七：AI岗位情报Agent

基于JD深度分析岗位能力要求、面试考察重点、成长路径建议。

### 核心创新点

1. **原创27种职场人格体系** — 非MBTI/DISC改制，自研命名体系
2. **人格驱动的多Agent协同** — 测试数据直接输入Agent，形成闭环
3. **前后端分离安全架构** — 前端Netlify + 后端Railway代理，API Key不暴露
4. **移动端优先+暗色模式** — 适配年轻用户使用习惯
5. **零代码可体验** — 无需依赖第三方平台，完全自建

## 技术架构 / AI能力及技术方案

### 前端技术栈

| 技术 | 用途 |
|------|------|
| HTML5 + CSS3 | 页面结构与样式 |
| Vanilla JavaScript | 全部前端逻辑，无框架依赖 |
| Canvas API | 15维雷达图绘制 |
| localStorage | 测试历史记录持久化 |

### 后端技术栈

| 技术 | 用途 |
|------|------|
| Node.js + Express | API服务器 |
| Railway | 后端部署 |
| DeepSeek API | AI推理引擎 |

### 数据流架构

```
用户作答 → 30题答案 → 15维计分(L/M/H) → 曼哈顿距离匹配 → 27种人格
                                                                  ↓
用户输入JD + 简历/面试回答 → callAI() → Railway /api/chat → DeepSeek API
                                                                  ↓
                                                            AI分析结果
```

### 人格匹配算法

1. 每道题A/B/C选项映射1/2/3分
2. 每个维度2道题，总分2-6
3. 总分2-3→L(低)，4→M(中)，5-6→H(高)
4. 15维L/M/H转为数字向量(L=1, M=2, H=3)
5. 与27个人格模板计算曼哈顿距离
6. 极端冲突(HvsL)加权重排序
7. 最高相似度<60%→兜底非标人人格

### AI能力实现

三个Agent均通过后端代理调用DeepSeek API：

- **简历Agent**：System Prompt引导AI按"JD分析→匹配度评分→缺失点→优化建议"结构输出
- **面试Agent**：AI根据JD实时生成5题 + 逐题评分（结构30%+内容30%+匹配度40%）
- **岗位Agent**：AI按"能力要求→面试重点→成长路径"三块输出

## 应用场景

### 场景一：大学生求职前自我认知

毕业季学生通过30题快速了解自己的职场人格类型，获取岗位推荐方向，降低择业试错成本。

### 场景二：简历投递前的匹配优化

求职者拿到目标岗位JD后，使用简历Agent进行匹配度分析，针对性优化简历，提高简历通过率。

### 场景三：面试前的实战练习

使用面试Agent进行AI模拟面试，不限次数，每次随机出题+即时评分反馈，积累面试经验。

### 场景四：跨行业转型的岗位调研

通过岗位情报Agent快速了解目标行业的能力要求和成长路径，辅助职业决策。

## 使用的模型

本作品使用**DeepSeek Chat**作为AI推理引擎。选型理由：

1. **中文能力突出** — 显著优于同价位模型的自然语言理解和生成能力
2. **性价比高** — API成本约0.5元/百万tokens，适合学生团队
3. **兼容OpenAI格式** — 与标准chat/completions接口一致，便于迁移
4. **支持长上下文** — 满足简历全文+JD的输入长度要求

API Key部署在Railway后端环境变量中，通过代理转发调用，前端不可见。

## 商业化思考

### 商业模式

```

人格测试免费            无限Agent使用           团队洞察报告
```

### 目标用户

- **C端**：应届毕业生、在校大学生、职场新人（1-3年经验）
- **B端**：高校就业指导中心、企业HR部门、招聘平台

### 收入模型

| 收入来源 | 说明 | 预计客单价 |
|---------|------|-----------|
| C端增值订阅 | Agent无限使用+高级功能 | 19.9元/月 |
| B端企业版 | 团队人格画像+人才评估 | 5000-20000元/年 |
| 高校合作 | 就业指导系统+数据报告 | 按项目收费 |

### 竞争壁垒

1. **人格体系壁垒** — 27种人格的命名、描述、维度模式串经过算法验证，模仿成本高
2. **数据飞轮** — 用户测试数据积累越多，匹配算法越准
3. **人格×AI双锁定** — 单纯的人格测试或AI工具都无法替代两者结合的体验
